⚙SuccessFactors Business Rules to Workbook Agent v1.2

Transforms SuccessFactors Business Rules CSV exports into a consolidated formatted Excel workbook — with natural-language IF/THEN/ELSE conditions, rule version history, and optimisation recommendations

? How to Prepare Your Input Files ▼

ZIP Business Rules Export Package

Package your SuccessFactors business rule exports into a single ZIP file. The ZIP may contain many files — only the four listed below are processed, in order:

  1. Rule.csv
  2. Rule-Used Message Definitions.csv
  3. Message Definition.csv
  4. Message Definition-Parameters.csv
⚠  Do not rename files inside the zip. Exact filenames are required for correct matching.
Other files in the ZIP are ignored safely. Files may be inside a subfolder within the ZIP.
CSV Rule.csv — Core Rules

Export business rules from SuccessFactors:

  1. Navigate to Admin Center → Configure Business Rules
  2. Click Export Rules (or use Import/Export Data)
  3. Select All Rules or filter by type / base object
  4. Download as CSV

Expected columns include: id, startDate, name, description, ruleType, baseObject, scenario, body

The tool auto-detects column names — check the log panel for the exact headers read from your file if the worksheet appears empty.
CSV Message Definition Files

Export message data linked to business rules:

  1. Navigate to Admin Center → Import and Export Data → Export Data
  2. Export Rule-Used Message Definitions
  3. Export Message Definition
  4. Export Message Definition-Parameters
  5. Add all three CSV files to the same ZIP as Rule.csv
These three files are optional. If absent, only the Business Rules worksheet is generated.
📄 Output Workbook Structure

The generated Excel workbook contains:

  • Table of Contents — Clickable links to every worksheet
  • Business Rules — One row per unique rule ID (latest version only), with natural-language IF / THEN / ELSE, Rule History, and Recommendations columns
  • Rule-Used Message Definitions — if file present in ZIP
  • Message Definitions — if file present in ZIP
  • Message Def - Parameters — if file present in ZIP
  • Version History — Tool changelog
The log panel shows all filenames found in the ZIP and the actual column headers from Rule.csv — useful for diagnosing any blank output.
