⚙SuccessFactors Config to Workbook Agent v1.19.1

Transforms SuccessFactors HRIS XML and MDF CSV/ZIP configuration files into a consolidated formatted Excel workbook — including Object Association hierarchy mapping

? How to Prepare Your Input Files ▼

XML HRIS · Legacy Foundation Objects

Upload the XML configuration file directly from SuccessFactors. This covers:

  • Legacy Foundation Objects — Location, Department, Division, Job Classification, Pay Grade, and other foundation data structures
  • Succession Data Model — XML-based succession configuration files
Accepted format: .xml
CSV MDF Generic Objects

Export the object definitions from your SuccessFactors instance:

  1. Navigate to Admin Center → Import and Export Data
  2. Select Export Data
  3. Set Generic Object to Object Definitions
  4. Choose Yes for all records, or select specific objects
  5. Download the generated CSV file
Accepted format: .csv  |  Skip if uploading a .zip in Step 2
ZIP CSV Object Associations (optional)

To generate the Association Hierarchy worksheet, use either method:

  1. Navigate to Admin Center → Import and Export Data → Export Data
  2. Method A — ZIP (recommended): Export Object Definitions with Include Dependencies = Yes. The zip covers Steps 1 and 2 in one upload
  3. Method B — CSV only: Export Object Definitions-Associations and upload the CSV here. Also upload the Object Definitions CSV in Step 1
⚠  Do not rename files inside the zip. The tool expects Object Definition-Associations.csv and Object Definition.csv.
